What is this project?

A slightly improved version of my first software project

This "Songify" project is - in a way - a remake of a project I have had to work on during my very first semester. It is my attempt to use newly gained knowledge about different design/programming concepts.
My version of the project is trying to achieve a straight forward way to browse musical artists of different genres of music.

This project is re-created using high-level web framework Django, and makes heavy use of Spotify API.
